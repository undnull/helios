# Getting started: Requirements
#### You need to have a basic knowledge about:
* Modern OpenGL (4.1+)
* GLFW ([GLFW: Getting Started](https://www.glfw.org/docs/latest/quick.html))
* C++17

#### You need to have this software on your machine:
* Git
* A GNU bash-compatible shell.
* A C++17 compiler.
* A C99 compiler (for dependencies)
* CMake 3.10 and higher
* _Optionally:_ [glslangValidator](https://github.com/KhronosGroup/glslang) and [SPIRV-Cross](https://github.com/KhronosGroup/SPIRV-Cross)
